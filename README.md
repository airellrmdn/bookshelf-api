# Bookshelf-API
## Introduction
A RESTful API built with Node.JS using Hapi for CRUD operation.

## Manual Installation

Clone the repo:

```bash
git clone https://github.com/airellrmdn/bookshelf-api.git
cd bookshelf-api
```

Install the dependencies:
```bash
npm install
```

Running the server:
```bash
npm run start
# or
npm run start:dev
```
